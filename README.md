# Week 1 - Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Online Retail II Dataset to understand customer purchasing behavior before developing a recommendation system.

---

## Objective

The objective of this project is to analyze customer transactions, identify data quality issues, discover sales patterns, and generate business insights using Python.

---

## Dataset

- Dataset: Online Retail II
- Period: 2009–2011
- Source: UCI Machine Learning Repository
- https://github.com/muskannazeer/OnlineRetail

---

## Tools & Libraries

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Tasks Performed

- Loaded the dataset
- Displayed dataset information
- Checked missing values
- Checked duplicate records
- Calculated revenue
- Top 10 products by quantity
- Top 10 products by revenue
- Country-wise sales analysis
- Monthly revenue trend
- Correlation heatmap
- Outlier detection using boxplots
- Business insights

---

## Business Insights

- United Kingdom generated the highest revenue.
- WHITE HANGING HEART T-LIGHT HOLDER was the highest-selling product by quantity.
- REGENCY CAKESTAND 3 TIER generated the highest revenue.
- Monthly revenue showed seasonal fluctuations.
- The dataset contains missing values and duplicate records.
